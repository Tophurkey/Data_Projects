My Projects

As part of the 2023 ASQ PRQC Conference I presented on Gen AI applications which could be used in Quality Engineering. This presentation details the current state of Generative AI, applications, and opportunities for use in quality engineering processes. The code repo contains multiple samples on applying Gen AI models to perform complex daily tasks of a quality engineer. Overall feedback from this presentation was positive and led to multiple invitations to continued presentations at various companies including Amazon.

My work in walking through the amazing book by Harrison Kinsley and Daniel Kukiela called "Neural Networks from Scratch in Python". This book walks you through writing a Neural Network using the bare minimum of libraries to help best understand the inner workings of a Neural Network. This project has been a personal goal of mine to help really understand the underpinnings of these amazingly complex LLM's and ML Models that are driving the biggest leaps in technology today.

Using MSSQL to create databases, tables, and views for analyzing COVID Data and practicing SQL programming. Includes the use of JOINS, CTE's, Temp Tables, and math to calculate statistical metrics.

Using Tableau Public to host a dashboard that utilizes a static version of the SQL data. These visualizations detail the current status of COVID and relations between certain metrics. This dashboard utilizes maps, tooltips, and multiple graphs to best understand the results of the initial data analysis from SQL.
